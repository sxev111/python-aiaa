# Python AI Assistant

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/sxev111/python-aiaa.git
   ```
2. Navigate into the project directory:
   ```bash
   cd python-aiaa
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Launch the application:
   ```bash
   python main.py
   ```
2. Follow the on-screen instructions to interact with the AI Assistant.

## Notes
- Ensure Python 3.6+ is installed on your machine.
- Refer to the [documentation](https://github.com/sxev111/python-aiaa/wiki) for more advanced usage and configuration options.
